## 简介

BLDebugTools是一个用于debug阶段调试的工具集合,目前提供了:

- FPS检测
- 沙盒文件预览与分享
- 有新的想法可以提交issue给我🎆🎆🎆

## 使用

- 导入

  - swift

    ```
    import BLDebugTools
    ```

  - objc

    ```
    #import<BLDebugTools/BLDebugTools.h>
    ```

- 使用

  ```swift
  DebugWindow.shared.begin()
  ```
